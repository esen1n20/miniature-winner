Политика конфиденциальности

Настоящая Политика определяет порядок обработки и защиты информации о пользователях сервиса Platinovskaya

1. Общие положения

1.1. Настоящая Политика конфиденциальности (далее — Политика) действует в отношении всей информации, которую онлайн-школа фитнеса «Platinovskaya» (далее — Сервис) может получить о Пользователе во время использования сайта platinovskaya.ru и его сервисов.

1.2. Использование Сервиса означает безоговорочное согласие Пользователя с настоящей Политикой и указанными в ней условиями обработки информации.

1.3. В случае несогласия с условиями Политики Пользователь должен прекратить использование Сервиса.

2. Собираемая информация

2.1. Сервис может собирать следующую информацию о Пользователе:

  • Фамилия, имя, отчество
  • Контактный телефон
  • Адрес электронной почты (e-mail)
  • Данные об использовании сервисов (история покупок, просмотренные курсы)
  • Технические данные (IP-адрес, тип браузера, операционная система)

2.2. Сервис защищает данные, которые автоматически передаются в процессе просмотра рекламных блоков и при посещении страниц.

3. Цели сбора информации

3.1. Информация о Пользователе используется в следующих целях:

  • Предоставление доступа к курсам и материалам
  • Связь с Пользователем для уточнения деталей заказа
  • Информирование о новых курсах, акциях и специальных предложениях
  • Улучшение качества сервиса и персонализация контента
  • Проведение статистических исследований
  • Обработка платежей

4. Защита информации

4.1. Сервис принимает необходимые организационные и технические меры для защиты персональной информации Пользователя от неправомерного доступа, уничтожения, изменения, блокирования, копирования, распространения.

4.2. Доступ к персональным данным имеют только уполномоченные сотрудники, которые обязаны соблюдать конфиденциальность.

4.3. Сервис не несет ответственности за безопасность данных, если утечка произошла вне систем Сервиса (например, при использовании Пользователем ненадежного пароля).

5. Передача данных третьим лицам

5.1. Персональные данные Пользователей могут быть переданы третьим лицам в следующих случаях:

  • Пользователь явно выразил согласие на передачу
  • Передача необходима для обработки платежей (платежные системы)
  • Передача необходима для выполнения условий договора с Пользователем
  • Передача предусмотрена российским законодательством

5.2. Сервис не продает персональные данные третьим лицам.

6. Файлы cookie

6.1. Сервис использует файлы cookie для обеспечения функциональности сайта, анализа трафика и персонализации контента.

6.2. Пользователь может настроить свой браузер таким образом, чтобы отклонять все или некоторые файлы cookie, однако это может повлиять на функциональность Сервиса.

7. Права Пользователя

7.1. Пользователь имеет право:

  • Получить информацию о своих персональных данных, обрабатываемых Сервисом
  • Требовать уточнения, блокирования или удаления своих персональных данных
  • Отозвать согласие на обработку персональных данных
  • Обжаловать действия Сервиса в уполномоченном органе по защите прав субъектов персональных данных

7.2. Для реализации своих прав Пользователь может направить запрос на электронную почту [email protected].

8. Изменение Политики

8.1. Сервис имеет право вносить изменения в настоящую Политику конфиденциальности.

8.2. Новая редакция Политики вступает в силу с момента её размещения на сайте.

8.3. Продолжение использования Сервиса после внесения изменений означает согласие с новой редакцией Политики.
